Reverse String Word Wise
Easy

Given a string, reverse the order of words in the string while maintaining the order of characters within each word.

Examples:
Example 1:

Input:
Hello World

Output:
World Hello

Explanation:
In the input string 'Hello World', reversing the order of words gives 'World Hello'.

Example 2:

Input:
LeetCode is fun

Output:
fun is LeetCode

Explanation:
In the input string 'LeetCode is fun', reversing the order of words gives 'fun is LeetCode'.

Example 3:

Input:
Coding is awesome

Output:
awesome is Coding

Explanation:
In the input string 'Coding is awesome', reversing the order of words gives 'awesome is Coding'.

Constraints:
The input string contains only printable ASCII characters and spaces.
Words in the string are separated by a single space.
The length of the string is at most 10^5 characters.
Optimize the algorithm for O(n) time complexity and O(1) space complexity, where n is the length of the string
